## [1.4.0] - 2026-03-25 (当前版本)

### ✨ 新增功能

- **股票收藏** - 研究和对比页面添加收藏功能
  - useFavorites Hook管理localStorage收藏
  - 支持添加/移除/切换收藏
  - 研究页面显示收藏股票快速入口
  - 对比页面股票卡片添加星标

- **最近搜索** - 搜索组件添加最近搜索记录
  - localStorage存储最近搜索(最多5条)
  - 聚焦搜索框时显示最近搜索
  - 自动保存搜索历史

- **研究完成通知** - 添加Toast通知系统
  - success/error/info/warning四种类型
  - 自动消失(4秒)或手动关闭
  - 可选提示音效

- **批量研究** - 新增批量研究组件
  - 同时研究多个股票(最多10个)
  - 从收藏快速添加
  - 研究页面模式切换入口

- **K线全屏** - K线图表支持全屏查看
  - 使用浏览器Fullscreen API
  - 全屏后自动调整图表大小

- **市场状态指示** - 监控页面显示开盘/休市状态
  - 根据A股交易时间自动判断(9:30-11:30, 13:00-15:00)
  - 每分钟自动更新

- **返回顶部** - 全局添加返回顶部按钮
  - 滚动超过400px后显示
  - 点击平滑滚动到顶部

- **全局页脚** - 添加包含免责声明的页脚
  - 显示数据来源信息

- **报告导出** - 研究报告支持导出
  - JSON格式完整数据
  - Markdown格式可读报告

- **复制和分享** - ReportCard添加操作按钮
  - 复制股票代码
  - Web Share API分享
  - 复制链接分享

- **API响应时间监控** - StreamingProgress显示执行时长
  - 总耗时计时器
  - 每个Agent步骤执行时长

- **历史记录管理** - 研究历史支持搜索和清空
  - 按股票代码/名称搜索过滤
  - 一键清空所有历史

- **快捷键提示优化** - 优化ShortcutHint样式
  - 更明显的键盘按键样式
  - 更好的视觉层次

## [1.1.0] - 2026-03-25
